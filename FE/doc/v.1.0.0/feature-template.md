## API 명세

### Endpoint 1: GET /api/feature

**Request:**
```typescript
{
  query: {
    param1: string;
    param2: number;
  }
}
```

**Response:**
```typescript
{
  status: 200,
  data: {
    result: FeatureData[];
  }
}
```

**Error Responses:**
- 400: Bad Request
- 401: Unauthorized
- 404: Not Found
- 500: Internal Server Error

### Endpoint 2: POST /api/feature

**Request:**
```typescript
{
  body: {
    field1: string;
    field2: number;
  }
}
```

**Response:**
```typescript
{
  status: 201,
  data: {
    id: string;
    message: string;
  }
}
```

## 에러 처리

### 에러 타입

#### 1. 네트워크 에러
- **발생 조건**: API 호출 실패
- **사용자 메시지**: "네트워크 연결을 확인해주세요"
- **복구 방법**: 재시도 버튼 제공

#### 2. 유효성 검증 에러
- **발생 조건**: 잘못된 입력값
- **사용자 메시지**: 필드별 에러 메시지
- **복구 방법**: 입력값 수정 안내

#### 3. 권한 에러
- **발생 조건**: 인증 만료, 권한 부족
- **사용자 메시지**: "다시 로그인해주세요"
- **복구 방법**: 로그인 화면으로 이동
